#Website for Bloom & Bean

This website is created is for a coffee shop business, to attract customers an help them interact with the coffee shop online.
The reason for creating the website is to promote the coffee shop,share information and make it easy for customers to view the menu and place order online as well as get their orders delivered to them.
In the website i included a home page, meanu page, about page, contact page, order form, gallery page, special page and navigation links to help users explore the coffee shop easily.
When opening the folder, you will first see a homepage whith the coffee shop name, images and links to other pages where you will view the menu,specials,be able to order and learn about the shop.
